# Day 4 — LLM Song Classification + Accuracy Tuning

Two major phases: BPM experiments and full LLM classification pipeline.

**BPM experiments:** Tested Essentia classical, TempoCNN neural, GPT-4o-mini, GPT-4o, Claude Sonnet, Claude Opus. Finding: smaller LLMs beat larger ones for BPM recall (memorized database values vs reasoning from genre). Strategy D (duration-verified YouTube) confirmed best audio source. "Database recall" prompt framing > "estimate" framing.

**Full classification:** 1,360 songs classified via GPT-4o-mini with provider-agnostic wrapper. Confidence-aware ensemble (LLM + Essentia hybrid). Switched BPM strategy from Essentia-primary to LLM-primary — Essentia's octave errors are catastrophic, LLM's errors are graceful.

**Accuracy tuning:** 48% → 83% product accuracy through: energy normalization fix, GRND gaussian narrowing, confidence-aware ensemble, LLM energy/acousticness fallback. Key lesson: bucket accuracy (66%) made us think we had a problem; product accuracy (83%) showed the system already worked.

579 → 621 tests passing.
