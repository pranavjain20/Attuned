# Day 8 — Classification Merge Fix + Validator

Fixed the Essentia/LLM merge pipeline. Key discovery: passing Essentia values as "hints" in the LLM prompt created an echo chamber — LLM parroted Essentia 96% of the time. In all 54 cases where the LLM independently disagreed, it was correct. Removed hints, let each source give independent values.

Added `essentia_energy` / `essentia_acousticness` columns for idempotent recompute — `recompute-scores` was reading from columns it also wrote to, producing different results on re-run.

Removed genre outlier validation (182/300 false positives — penalized legitimate musical diversity).
